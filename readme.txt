Readme For FAT32 WEB Parser

How TO USE?
0. FAT32 Web Parser is for Python3.x. Move directory to where run.pyc is placed
1. Every python file is compiled(Except sources dir) and you need to just run run.pyc file!
2. Connect http://127.0.0.1:8080 by Your Browser
3. Attatch Your FAT32 Disk Img file. We only Offer RAW data Analysis.
4. Please wait until uploading and parsing your image.
5. After Parsed you can see alert that says parsing is finished.
6. You can explore Your Disk Img File.
7. You can move to other directory or carve file by clicking row of table.
8. When you finish using, please Click Remove IMG to delete your Cache DATA
